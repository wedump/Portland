* 미디엄 참고

> 일단 기본기능 구현  
  - 폰트, 크기, 색상, 블록색상, 볼드, 이태릭, 밑줄, 줄긋기(지우기줄), 정렬(일반<->가운데)

> 이후 기능
  - 리스트, 테이블, 링크, 이미지, 동영상, 백그라운드 등

> 아이디어
  - 글의 단락 나누는 기능, 레이아웃 잡기, 신문 처럼
  - 동일 글을 단락을 다누고, 서로 길드원끼리 각각 담당하여 동시에 작성할 수 있는 기능
  - 단락 내 다양한 테마/하이라이팅 지원
  - 사용자가 작성한 컴포넌트(HTML/CSS/JS 로) 적용
  - 템플릿(레이아웃, 테마, 하이라이팅, 컴포넌트, 시그니처 마크 등) 작성/적용/공유(마켓)
  - 마크다운(일부) + 위지윅 : 마크다운->위지윅 자동변환
  - 본인 시그니처 마크 넣기(카피라이트 등 포함)
  - 말풍선 지원
  - 복사/붙여넣기 이쁘게(이미지, 글, 링크 등)
    꾸며 넣기
    다른 사이트의 HTML/CSS 그대로 가져다 넣기 등(예, Gist 복사/붙여넣기하면 그대로 나온다)
  - 캘라그라피 같은 느낌(컨셉/감성)
    예) 형광펜 하이라이트(블록색상 기능이지만 손으로 한것처럼)

> 역할
  - 에디터 : 단락/메뉴 관리(배치 등)
  - 단락 : 글을 쓰고 수정할 수 있다
  - 메뉴 : 버튼을 가지고 있다(버튼들의 모음)
  - 버튼
    1) 누르면 선택된(블록) 글을 자신이 의미하는 기능(사이즈조절, 폰트변경 등)에 맞게 수정할 수 있도록 단락들에 요청
    2) 선택된 글이 없는 경우, 자신의 기능을 현재상태에 반영하여 새로 쓰여지는 글들에 기능이 적용되도록 단락들에 요청

> 속성 및 행위
  - 에디터
    1) 속성 : 단락/메뉴 리스트
    2) 행위 : 단락/메뉴 추가, 변경, 삭제, 그리기 등
  - 단락
    1) 속성 : 플레이스홀더, 사이즈
    2) 행위 : 자신이 포함하는 글을 수정할 수 있는 기능들 제공
  - 메뉴
    1) 속성 : 버튼 리스트
    2) 행위 : 버튼 추가, 그리기
  - 버튼
    1) 속성 : 툴팁, 이미지, 사이즈
    2) 행위 : 자신이 의미하는 기능(사이즈조절, 폰트변경 등)에 맞게 단락들에게 선택된 글의 수정을 요청
